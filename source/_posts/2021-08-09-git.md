# SSH keys和Tokens
## 创建SSH keys
1. 创建SSH keys,下列命令是创建4096位的RSA密钥。
```bash
ssh-keygen -t rsa -b 4096 -C "your_email"
```
2. Windows端，在`用户名/.ssh`文件夹中打开`.pub`文件。
3. 复制SSH key到GitHub Settings-->SSH and GPG keys-->New SSH keys。
## tokens
1. 在GitHub Settings-->Developer settings-->Personal access tokens-->Tokens (classic)-->Generate new tokens (classic)创建一个token。
2. 请务必立即复制您的个人访问令牌。您将无法再次查看它！
# 提交代码
```bash
git init

git config user.name "you"
git config user.email "you@users.noreply.github.com"

git remote add origin git@github.com:you/your_repo.git

git add .
git commit -m "Initial commit"
git branch -M main
git push -u origin main
```
# 仓库分支
## 创建分支
```bash
git branch -a
git branch dev
git checkout dev
```
## 合并分支

```bash
git branch -a
git branch dev
git checkout dev

git checkout main
git pull origin main
git merge dev
git push origin main
```
## 删除分支
```bash
git push origin --delete dev
git branch -D dev
```
# 其它命令
## 关闭自动换行
```bash
git config core.autocrlf false
```
## 下载指定分支
```bash
git clone -b branch_name https://github.com/user/repo.git	
```
